node-staticServer
=================
> nhs(Nod.js HTTP Server) is a node.js http server.
##Getting Started
```shell
npm install -g nhs
```
##Usage
Starting nhs
nhs server.js
##Available Options
 -h, --help     output usage information<br />
 -V, --version  output the version number<br />
 -d, --dev      launch a server in the development mode
